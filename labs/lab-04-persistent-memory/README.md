# Lab 04 — Persistent Memory

**Outline:** Persistent Memory  
**Workflow stage:** Plan  
**Wow:** ปิด–เปิด session แล้วยังจำ preference

## Goal

ให้ agent จำ preference โปรเจกต์ (เช่น สไตล์ UI, ห้ามแก้ backend จาก FE agent) ข้าม session

## Steps

1. บันทึก preference ใน memory / project notes ตามเครื่องมือที่ใช้
2. ปิด session แล้วเปิดใหม่
3. สั่งงานที่ต้องใช้ preference โดยไม่บอกซ้ำทั้งหมด

## Definition of Done

- [ ] Preference ถูกเรียกใช้หลังเปิด session ใหม่
- [ ] เปรียบเทียบสั้น ๆ: built-in memory vs ไฟล์ใน repo
- [ ] learning-log มี before/after
