# Lab 4: Memory

**เวลา:** 30 นาที  
**เครื่องมือ:** Claude Code และ/หรือ OpenCode  
**นอก trip contract flow** — ไม่บังคับสร้าง JSON contract ใหม่

## เป้าหมาย

ให้ Agent จำ preference ทริปข้าม session แล้วเปิดใหม่แล้วยังใช้ได้

## ไฟล์ในโฟลเดอร์นี้

- `memory-brief.md` — ข้อมูลที่ต้องจำอย่างน้อย

## ขั้นตอน

1. อ่าน `memory-brief.md`
2. บันทึกข้อมูลลง memory/project notes ของเครื่องมือที่ใช้ (ตามวิธีของ Claude Code / OpenCode)
3. ปิด session แล้วเปิดใหม่
4. สั่ง Agent อ้าง preference ที่จำไว้ (เช่น ปรับ dining tags หรือทบทวน trip brief)
5. บันทึกวิธีที่ใช้และผลใน `workspace/learning-log.md`

## เกณฑ์ผ่าน

- [ ] จำได้อย่างน้อย: งบ ≤ 3,000 / อาหารไทย / คู่รัก 2 คน / กลับสีลมก่อน 20:00
- [ ] เปิด session ใหม่แล้วยังอ้างข้อมูลได้
- [ ] จดวิธี memory ที่ใช้ใน learning-log
