# Permission notes (Lab 03)

ค่าแนะนำสำหรับห้องเรียน:

- Deny: `git push` (ทุก remote)
- Deny: ลบโฟลเดอร์ `shared/` ทั้งก้อน
- Ask: คำสั่งที่แก้ `.env` หรือลบ `workspace/` ทั้งก้อน

ไฟล์อ้างอิง: `.claude/settings.json` ที่ root ของ repo

หลังทดสอบ ให้จดใน learning-log:

1. คำสั่งที่ลอง
2. ผล (deny / ask / allow)
3. ทำไมกฎนี้สำคัญกับทีมหลาย agent
