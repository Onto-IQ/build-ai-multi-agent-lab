# Lab 02 — OpenCode Specialist + เทียบ Claude Code

**Outline:** OpenCode Setup & Multi-model  
**Workflow stage:** Plan  
**Wow:** ตารางเทียบสิทธิ์/โมเดลคนละเครื่องมือใน 1 หน้า

## Goal

ติดตั้ง/ตั้งค่า OpenCode สร้าง specialist แล้วเปรียบเทียบกับ Code Reviewer จาก Lab 01

## Steps

1. ตรวจ `opencode` ใน Terminal (Task: `env-checklist`)
2. สร้าง agent ใน `.opencode/agents/` หรือผ่าน `opencode agent create`
3. ใช้ multi-model ถ้าห้องจัดให้ (ดู `.env`)
4. เขียนตารางเทียบใน `workspace/learning-log.md`

## Definition of Done

- [ ] OpenCode agent ทำงานได้ในโปรเจกต์นี้
- [ ] มีตารางเทียบ Claude Code vs OpenCode (อย่างน้อย 4 แถว: model, permission, multi-agent, cost)
- [ ] learning-log อัปเดต
