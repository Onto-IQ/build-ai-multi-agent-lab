# Lab 09 — Synthesizer Agent

**Outline:** The Synthesize Pattern  
**Workflow stage:** Test → Ship  
**Wow:** รวม FE+BE แล้วเปิดดู/รันได้จริง

## Goal

สร้าง Synthesizer รวมผล Frontend + Backend ให้เป็นแอปที่ตรวจการทำงานได้

## Steps

1. ตรวจว่า FE และ BE พร้อมตาม ownership
2. ให้ Synthesizer รวม/เชื่อม (ไม่แย่งเขียนคนละโฟลเดอร์เกินสิทธิ์)
3. เปิด `apps/sample-dashboard/frontend/index.html` หรือ static server สั้น ๆ แล้วจับภาพ/จดผล

## Definition of Done

- [ ] มี synthesizer prompt หรือ agent
- [ ] แอปแสดงสถานะจาก backend ได้ หรือมีบันทึกว่าทำไมยังไม่ได้พร้อมแผนแก้
- [ ] Aggregation notes ใน learning-log
