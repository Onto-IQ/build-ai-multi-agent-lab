# FLUX-SETUP — บอร์ดงาน Agent Cost Board

Flux เป็น**ชั้นมอบหมายงาน**ของโปรเจกต์นี้ ไม่ใช่บอร์ดเดโม่คนละเรื่อง

## สิ่งที่ต้องได้เมื่อจบคู่มือนี้

1. มีบอร์ดสำหรับ Agent Cost Board
2. คอลัมน์: Interview → Plan → Build → Test → Ship
3. การ์ดอย่างน้อย 3 ใบที่ตรงงานแอป
4. (ถ้าห้องใช้) เชื่อม MCP กับ Claude Code / VS Code

## ขั้นตอนแนะนำ

1. สมัคร/เข้า Flux ตามที่วิทยากรแจก  
2. ใส่ `FLUX_API_KEY` และ `FLUX_WORKSPACE_URL` ใน `.env`  
3. สร้างบอร์ดชื่อ เช่น `Agent Cost Board — <ชื่อผู้เรียน>`  
4. ตั้งคอลัมน์ 5 ขั้นตามลำดับงานคอร์ส  
5. สร้างการ์ดตัวอย่าง:
   - Frontend: ปรับการ์ดยอดใช้ / ตารางรอบ
   - Backend: เติม runs.json จากรอบ Lab
   - QA: รัน gate-quality แล้วเลื่อนไป Test/Ship
6. มอบหมายผู้รับผิดชอบเป็นบทบาท + เครื่องมือ (Claude Code / OpenCode)
7. ทดสอบเลื่อนการ์ดหนึ่งใบ

## ความสัมพันธ์กับชั้นอื่น

| ชั้น | หน้าที่ |
|---|---|
| Flux | มองเห็นงาน + มอบหมาย |
| Agent Cost Board | แสดงต้นทุน/สถานะทีมตัวเอง |
| JSON ใน workspace | แหล่งความจริงส่งงานต่อ |

## หลังจบคอร์ส

สร้างบอร์ดใหม่สำหรับงานอื่น → มอบหมาย Claude + OpenCode → ใช้สัญญา + ด่านสูตรเดิม
