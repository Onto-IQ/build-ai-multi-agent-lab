# Lab 10 — Kanban Collaboration (Flux required)

**Outline:** Collaboration Layer (Kanban board)  
**Workflow stage:** ทั้งเส้น Interview → Ship  
**Wow:** การ์ดขยับตอนมอบหมายงานให้ agent

## Goal

ตั้ง **Kanban Board** ที่คนทั่วไปเข้าใจได้ และมอบหมายงานให้ agent

### ค่าเริ่มต้น (July 2026): **Flux**

- เชื่อม MCP กับ Claude Code / VS Code ตามคู่มือ Flux
- คอลัมน์แนะนำ: Interview · Plan · Build · Test · Ship

### ทางเลือกเปรียบเทียบ

- Multica (self-host/Docker) — ไม่บังคับ cloud
- AgentOps / Ruflow — observability / loop spotting (พูดสั้น ๆ)

## Definition of Done (บังคับมี Kanban)

- [ ] มีบอร์ดจริง (Flux แนะนำ) พร้อมคอลัมน์ workflow 5 ขั้น
- [ ] มีอย่างน้อย 3 การ์ดมอบหมายให้ agent/role
- [ ] อธิบายได้ว่าบอร์ดไม่ใช่แหล่งความจริงเดียว — contract/ownership มาก่อน
- [ ] learning-log มี screenshot หรือลิงก์บอร์ด
